# Тестовое задание 10X

Адаптивный каталог онлайн-курсов, разработанный на HTML, SCSS и чистом JavaScript.

## Возможности

- Адаптивная сетка карточек для экранов от 1920 до 320 пикселей
- Живой поиск курсов по названию
- Фильтрация по категориям с отображением количества курсов
- Именование классов по методологии БЭМ
- Семантическая HTML-разметка
- Поддержка управления с клавиатуры

## Локальный запуск

Откройте файл `index.html` в браузере.

Для разработки и автоматической компиляции SCSS:

```bash
npm install
npm run watch
```

## Сборка проекта

```bash
npm run build
```

Команда скомпилирует `styles/style.scss` в `styles/style.css`.

## Структура проекта

- `index.html` — основная HTML-разметка
- `styles/style.scss` — исходные стили
- `styles/style.css` — скомпилированные стили
- `scripts/script.js` — данные курсов, поиск и фильтрация
- `images` — изображения карточек и SVG-иконки

## Использованные технологии

- HTML5
- SCSS
- JavaScript (ES6+)
- БЭМ
